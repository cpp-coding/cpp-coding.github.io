模板-快速幂||取模运算

2020-04-08 模板

模板系列-快速幂||取模运算

题面

原题链接

题目描述

给你三个数 a,b,ka,b,ka,b,k ,求aba^b%kab

输入格式

输入只有三个整数 a,b,ka,b,ka,b,k

输出格式

输出一行一个字符串 a^b mod k=s，其中a,b,ka,b,ka,b,k分别为题目给定的值， s 为运算结果。

算法讲解

AC Code

#include <bits/stdc++.h>
using namespace std;
int main(){
    long long ans=1,i,j,k,m,n,b,p;
    scanf("%lld%lld%lld",&b,&m,&p);
    printf("%lld^%lld mod %lld=",b,m,p);
    while(m>0){
        if(m%2==1)
            ans=ans*b%p;
        b=b*b%p;
        m=m>>1;
    }
    printf("%lld",ans%p);
    return 0;
}
